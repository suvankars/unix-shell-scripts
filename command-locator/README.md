# Whichs 
####The Commands Locator

##### Description :
     A shell script (whichs) which locates a command.

###### Usage: 
`$ whichs [-a] command-name`
 `-a` switch is optional - If supplied, print all matching pathnames of each argument


###### Sample Input
`$./whichs -a git bash`

###### Sample Output
$ /usr/bin/git  
$ /bin/bash
